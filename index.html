🧠 ANKI - Para Memorização Eficiente

1

Baixar e Instalar

Acesse apps.ankiweb.net e baixe a versão para seu computador.

2

Importar as Palavras

Abra o Anki → "Import File" → Escolha o arquivo semana1.csv → Configure as colunas: Front (word), Back (meaning_pt), Extra (exemplos).

3

Estudar

Use os botões: Again (novamente), Good (bom), Easy (fácil). O sistema agenda revisões automaticamente.

🎮 QUIZLET - Mais Visual e Interativo

1

Criar Conta

Acesse quizlet.com e crie sua conta gratuita.

2

Criar Conjunto de Estudo

Clique em "Create" → "Study set" → Nomeie "Fluência 7 - Semana 1".

3

Importar e Estudar

Use "Import from Word, Excel..." → Cole o conteúdo do CSV → Configure Term (word) e Definition (meaning_pt). Use os modos: Flashcards, Learn, Write, Spell, Test, Match.

📊 GOOGLE SHEETS - Mais Simples

1

Acessar Google Sheets

Acesse sheets.google.com e faça login com sua conta Google.

2

Importar os Dados

File → Import → Upload → Selecione o arquivo semana1.csv → Configure separador como "Comma" → Import data.

3

Estudar

Cubra a coluna F (significado) e tente lembrar, use a coluna D (pron_pt) para pronúncia, crie filtros para organizar.

💡 Dicas Importantes

  • • Estude 7 palavras por dia (como no método)
  • • Use a coluna pron_pt para pronunciar corretamente
  • • Pratique os exemplos das colunas G e H
  • • Revise palavras antigas regularmente
  • • Mantenha consistência - melhor 20 min/dia

🆘 Problemas Comuns e Soluções

"Não consigo importar o arquivo"

Verifique se o arquivo está em formato CSV e se está usando vírgula como separador.

"As palavras não aparecem corretamente"

Verifique a codificação do arquivo (deve ser UTF-8). No Google Sheets, use "File" → "Import" em vez de "Open".

"Esqueci as palavras rapidamente"

Use a repetição espaçada (Anki é melhor para isso). Revise no mesmo dia, depois de 1 dia, 3 dias, 1 semana.
